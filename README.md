# Photography Portfolio  
### A growing collection of my work as a hobbyist photographer

This repository hosts the source code and content for my personal photography portfolio website.  
It serves as a space to showcase my photos, document my creative journey, and continuously add new work as I keep shooting and exploring different styles.

---

## About This Project

This portfolio is a long‑term, evolving project where I publish selected photographs from my personal collection.  
The goal is to create a clean, minimal, and visually engaging online gallery that reflects my growth as a photographer.

---

## Future Plans

- Add new photo collections regularly  
- Improve layout and visual presentation  
- Experiment with themes, galleries, and interactive elements  
- Optimize images for web performance
